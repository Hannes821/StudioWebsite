# Feldfeste Games Studio Website

Official landing page for **Castellum – Warfare & Cultivation**, hosted through
GitHub Pages at [feldfeste-games.com](https://feldfeste-games.com/).

## Structure

- `index.html` – page content and metadata
- `style.css` – responsive layout and visual styling
- `images/` – logos, concept material and optimized web assets
- `CONTENT_BACKLOG.md` – approved ideas for future website sections
- `CNAME` – custom GitHub Pages domain
- `robots.txt` and `sitemap.xml` – search-engine discovery

The site is deliberately framework-free. It can be previewed with any local
static file server.

## Content still required before publication

- Final Steam store URL
- Imprint details
- Privacy policy
- Direct contact details
- Key Features with manuscript-style icons
- Gameplay screenshots and a public roadmap when approved

## Publishing

Changes pushed to the `main` branch are published by the existing GitHub Pages
setup. Review the desktop and mobile layouts before pushing production changes.
